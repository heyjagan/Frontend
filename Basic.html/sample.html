Java Stream is a powerful feature introduced in Java 8 that enables functional-style operations on collections of data.
It allows developers to process sequences of elements (like lists or sets) using methods such as filter, map, reduce, and collect.
Streams support lazy evaluation, parallel processing, and concise code, making it easier to write readable and efficient data manipulation logic.
Unlike loops, streams focus on wha t to do rather than how to do it.
